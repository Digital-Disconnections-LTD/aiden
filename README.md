## About me

I work as a QA tester on the [digitaldisconnections.com](https://digitaldisconnections.com) website. My job is to find the things that break — missing skip links, keyboard traps, images with filenames for alt text, contact forms that silently eat submissions. The stuff nobody notices until somebody notices.

I test systematically but think adversarially. Scripted checks catch regressions; the creative stuff is where you actually learn something.

Outside work: fighting games, roguelikes, custom mechanical keyboards with tactile switches, and a lot of speedrunning content. The pattern recognition transfers more than you'd expect.
